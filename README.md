# Default-of-Credit-Card-Clients-Project
### Used Default of Credit Card Clients Dataset from UCI Repository
### Dataset Link: https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients
### Used decision trees, random forest, adaboost, and XGBoost models to predict whether clients would default on their credit card payments dpeending on different factors.
### Acheived test accuracy of 80.7% using XGBoost
