# Default of Credit Card Clients - Predictive Modeling Project
### Used Default of Credit Card Clients Dataset from UCI Repository
###### Dataset Link: https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients
### Preprocessed the dataset, extracted meaningful features to use as predictors in the final model.
### Used decision trees, random forest, AdaBoost, and XGBoost models to predict whether clients would default on their credit card payments depending on different factors.
### Acheived best test accuracy of 80.7% using XGBoost.
